# Дашборд мониторинга KPI и нагрузки управления

React-приложение для мониторинга загрузки, продуктивности и рисков сотрудников бэк-офиса (10–30 человек): документооборот, кадровое администрирование, юридическое сопровождение, закупки и договоры, архивное хранение, отчётность.

Реализовано по [PRD](../prd.md) v1.0.

## Разделы дашборда

- **Обзор** — сводные показатели, топ-3 сотрудника недели, динамика загрузки/продуктивности, приоритетные риски, направление для оптимизации
- **Нагрузка** — реестр фактической загрузки каждого сотрудника с визуальной шкалой и статусом
- **Продуктивность** — рейтинг по KPI, выполнению задач, качеству, просрочкам
- **Риски и процессы** — карточки сотрудников в зоне риска, анализ направлений деятельности
- **Реестр сотрудников** — редактируемая таблица: добавление, редактирование и удаление сотрудников с мгновенным пересчётом показателей во всех разделах

## Технологии

React 19 · TypeScript · Vite · Tailwind CSS v4 · Recharts · lucide-react

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Данные

На текущем этапе (v1.0) данные — демонстрационные, in-memory (не персистентны между сессиями). Реальные значения вводятся вручную через раздел «Реестр сотрудников».
